# Programming Club Frontend

React frontend for Programming Club Management System.

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Features

- Authentication (Login/Register)
- Role-based dashboards:
  - Manager Dashboard
  - Curator Dashboard
  - Student Dashboard
- Real-time collaborative code editing using WebSocket
- Task management
- Team management
